AI提问的艺术 · 第1课
为什么你的提问，AI听不懂？
"帮我写一个餐饮短视频文案"
这句话对AI来说，信息量 = 零
什么餐饮？ 火锅？咖啡？路边摊？
写给谁？ 学生？白领？宝妈？
什么目的？ 引流？品牌？促销？
什么平台？ 抖音15秒？小红书1分钟？
什么风格？ 搞笑？走心？科普？
有效提问的6个维度
① 角色 —— 告诉AI"你是谁"
② 背景 —— 告诉AI"我是谁"
③ 目标 —— 告诉AI"要什么结果"
④ 参考 —— 告诉AI"照着什么做"
⑤ 约束 —— 告诉AI"不能碰什么"
⑥ 格式 —— 告诉AI"怎么交付"
有效提问 = 角色 + 背景 + 目标 + 参考 + 约束 + 格式
每多给AI一个维度的信息，输出质量提升一个等级

AI提问的艺术 · 第1课
为什么顶级提问能出爆款？逐项拆解
1. 角色升级
不是"编导"→而是"操盘过张亮麻辣烫"的老手。让AI调用更高质量的知识库
2. 数据锚点
月流水8万、客单18元、复购60%。有数据AI才不编，没数据AI只能猜
3. 量化目标
"3天50组到店"比"引流"具体100倍。目标越具体，AI策略越针对
4. 结构拆解
直接给四段式+每段秒数。AI不用猜结构，精力全花在内容上
5. 场景约束
方言混搭+汤的特写。这些细节约束让内容有地气、有辨识度
6. 输出格式
要求分镜+台词+时长+字幕。拿到就能拍，不需要二次加工
第1课公式：角色 + 背景 + 目标 + 参考结构 + 约束 + 输出格式 = 可直接使用的文案
AI提问的艺术 · 第1课
常见踩坑 · 小白最容易犯的4个错误
错误1："帮我写一个好的文案"
正确："帮我写一条30秒抖音文案，目标是引导到店"
原因："好的"对AI没有定义。"30秒""抖音""到店"才是有效信息
错误2：一次问太多 "帮我写文案+拍摄方案+投放策略+运营计划"
正确：一次只要一个输出，分步骤追问
原因：一次要太多，每个都不深。先要文案，满意后再追问投放
错误3：不给数据 "我们家面很好吃，价格实惠"
正确："客单价18元，复购率60%，12小时慢熬骨汤"
原因："好吃""实惠"是你的感觉，AI需要事实和数据
错误4：拿到结果就用，不迭代
正确：第一次输出后追问"开头不够吸引人，换成反转型开头"
原因：顶级高手的秘密是"追问"。第一次输出是草稿，追问3次才是成品
课后练习
用6维框架，为你自己的店（或常去的一家餐饮店）写一段顶级提问。然后把AI输出发到群里，我们一起点评。
评分标准：是否包含全部6个维度？数据是否具体？输出格式是否明确？

普通人提问
帮我写一个卖理财课的短视频文案。
高手提问
你是知识付费领域的短视频运营。 我有一门"小白理财入门课"，价格99元，主要教基金定投和记账方法，目标用户月薪5000-10000的上班族。 写一条45秒抖音文案，目的引导购买，风格扎心但给希望。
顶级高手提问
你是帮长投学堂、简七理财做过百万GMV的知识付费操盘手。 【R·角色】以"过来人"身份，人设：从月光族到3年存下20万的普通上班族。 【E·参考】@巫师财经的"故事+数据"叙事 + @她理财的"闺蜜聊天"口吻。结构："扎心现实→数据冲击→转折故事→方法预告→限时引导" 【C·背景】"30天小白理财计划"99元/30节，记账→基金→保险，5000+学员，好评率97%。 【I·目标】抖音45秒，300+订单（2%转化率=1.5万播放），小黄车成交。 【P·边界】不承诺收益（违规）；不用"躺赚""暴富"；有免责声明；用共情不用说教；数据模糊化处理。 【E·输出】完整文案 + 3标题 + 评论区5条预埋 + 小黄车商品标题15字内 + 投放建议。
